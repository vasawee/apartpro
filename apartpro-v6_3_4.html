☁️ Cloud Sync - Firebase

✅ เซิร์ฟเวอร์ขึ้นระบบอย่างสมบูรณ์ | ทำให้ข้อมูลซิงค์อัตโนมัติระหว่างอุปกรณ์

📖 วิธีตั้งค่า Firebase (ปลอดภัย) — คลิกเพื่อดู
1. สร้าง Firebase Project
  • ไปที่ console.firebase.google.com
  • สร้าง Project ใหม่
  • เปิด Realtime Database และ Authentication
2. ตั้งค่า Security Rules (สำคัญ!)
{
  "rules": {
    "users": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
3. สำคัญ: ห้ามให้ Public Access
  • ⛔ ห้าม: rules = true
  • ✅ ต้อง: ตั้ง authentication และ .read/.write rules
สถานะการเชื่อมต่อ: ⏳ รอการตั้งค่า...
💾 สำรองข้อมูล

📥 ดาวน์โหลดข้อมูลทั้งหมดเป็น JSON | 📤 นำเข้าข้อมูลจากไฟล์ Backup

📅 Backup ล่าสุด ยังไม่เคยสำรอง
📊 ข้อมูลทั้งหมด คำนวณ...
🔄 สำรองข้อมูลอัตโนมัติ

เก็บสำรองขึ้น Firebase คลาวด์ทุกชั่วโมงโดยอัตโนมัติ (ต้อง Login Firebase ไว้)

☁️ สำรองขึ้นคลาวด์ (Firebase)

ปลอดภัยกว่า backup ในเครื่อง — กู้คืนได้แม้เครื่องพังหรือล้างเบราว์เซอร์ · เก็บประวัติย้อนหลัง 30 วันล่าสุด

สำรองขึ้นคลาวด์ล่าสุด: ยังไม่เคยสำรอง
📜 ประวัติการทำงาน (Audit Log)

บันทึกการลบห้อง ยืนยัน/ลบรายการชำระเงิน และปิด-เปิดบัญชีเดือน เพื่อตรวจสอบย้อนหลัง (เก็บ 500 รายการล่าสุด)

🔔 แจ้งเตือนการแจ้งซ่อม

รับแจ้งเตือนทันทีเมื่อผู้เช่าส่งคำแจ้งซ่อม

🔐 ความปลอดภัย

✅ เข้ารหัส Sensitive Data | 🔒 ป้องกัน XSS | 🛡️ ตรวจสอบข้อมูล

✅ การเข้ารหัสข้อมูล
  • 📱 เบอร์โทรศัพท์: เข้ารหัส AES-256
  • 🆔 เลขบัตรประชาชน: เข้ารหัส AES-256
  • 🏦 เลขบัญชีธนาคาร: เข้ารหัส AES-256
✅ ตรวจสอบข้อมูล
  • 📅 วันที่: รูปแบบ YYYY-MM-DD
  • 💰 จำนวนเงิน: ต้อง > 0
  • 🚪 หมายเลขห้อง: ไม่ให้ซ้ำกัน
  • 📞 เบอร์โทร: 10 หลัก, 08x-xxx-xxxx
⚠️ อันตรายจาก Resets

การรีเซ็ตจะลบข้อมูลทั้งหมด และไม่สามารถกู้คืนได้

การแจ้งเตือนผ่าน LINE

⚠️ LINE Notify ถูกปิดให้บริการแล้วตั้งแต่ 31 มี.ค. 2025 — ระบบนี้ใช้ LINE Messaging API ผ่าน Cloudflare Worker ที่ตั้งค่าแยกต่างหากแทน

💡 วิธีหา LINE User ID ของผู้เช่าแต่ละห้อง: ตั้งค่า LINE User ID ของผู้เช่าได้ที่หน้า "ห้องพัก" → แก้ไขห้อง → ช่อง "LINE User ID ของผู้เช่า" — ระบบจะส่งแจ้งเตือนค่าเช่า/แจ้งซ่อม/ใบเสร็จไปยังห้องที่กรอก LINE User ID ไว้เท่านั้น
